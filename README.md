# Student Performance Prediction using Machine Learning

## Overview
This project predicts student exam scores based on academic and lifestyle factors.

## Features
- Data preprocessing
- Feature encoding
- Model comparison
- Hyperparameter tuning
- Feature importance analysis
- Streamlit web application

## Machine Learning Models
- Linear Regression
- Random Forest
- Gradient Boosting
- XGBoost

## Tech Stack
Python, Scikit-learn, XGBoost, Streamlit, Pandas, NumPy

## Run the Project
1. Train the model
python src/train.py

2. Run the Streamlit app
streamlit run app/app.py
